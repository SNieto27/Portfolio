# Delivery Hero: Plataforma de Pedidos y Entregas 
"Delivery Hero" fue un proyecto web inspirado en plataformas como Uber Eats, diseñado para conectar a los consumidores finales con establecimientos de comida. Como intermediario, la plataforma gestiona todo el ciclo de pedido: desde la colocación del pedido, el cobro y la facturación, hasta el seguimiento y la entrega en la puerta del cliente. 

El proyecto contó con dos versiones: una interfaz para los clientes, donde podían explorar el menú y realizar sus pedidos, y otra para los motoristas, proporcionando toda la información necesaria para recoger y entregar los pedidos de manera eficiente. La aplicación fue desarrollada utilizando una base de datos no relacional en MongoDB, junto con tecnologías como HTML, CSS, JavaScript, NodeJS y Mongoose. 

Mi principal rol en este proyecto fue la implementación de la lógica backend y la integración con la base de datos, asegurando que la información fluyera de manera segura y eficiente entre todas las partes involucradas. También colaboré en el diseño de las interfaces de usuario, garantizando una experiencia intuitiva tanto para los clientes como para los motoristas.
